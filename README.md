# 自适应图片比例边框处理软件 (Adaptive Glass) V1.01

Adaptive Glass 是一款智能图片处理工具，专为摄影师和社交媒体内容创作者设计。它能自动识别图片内容，智能填充模糊背景以适应目标比例（如 Instagram 1:1, YouTube 16:9），并提供专业的水印和边框效果。

## 主要功能

### 1. 智能比例适配

- **自适应填充**: 自动计算图片与目标比例的差异，智能生成模糊背景填充空白区域。
- **多种预设比例**: 支持 1:1 (Instagram), 4:3, 3:2 (全画幅), 16:9 (YouTube), 9:16 (TikTok), 2.35:1 (电影感) 等。
- **内容缩放**: 支持调整原图在画布中的缩放比例 (50% - 100%)。

### 2. 高级背景处理

- **动态模糊**: 使用高斯模糊算法生成自然的背景效果。
- **风格选择**: 提供标准模糊、暗色玻璃、亮色玻璃三种风格。
- **亮度调节**: 支持背景亮度微调 (-100 到 +100)，让背景更契合主体氛围。

### 3. 精致边框系统

- **样式丰富**: 支持无边框、细边框、圆角边框。
- **参数可调**: 边框宽度、圆角大小、阴影强度、边框颜色（黑/白）均可自由调节。

### 4. 智能水印模块 (V1.01 还原增强版)

- **EXIF 数据读取**: 自动读取照片的相机型号、光圈、快门、ISO、焦距等参数并生成水印。
- **九宫格定位**: 支持水印在画面的 9 个方位（上/中/下 x 左/中/右）自由布局。
- **智能排版**:
  - **追加模式 (Append)**: 显示相机型号 + EXIF 参数 + 自定义文本。
  - **替换模式 (Replace)**: 仅显示自定义文本。
  - **备用模式 (Fallback)**: 优先显示 EXIF，无 EXIF 时显示自定义文本。
- **字体支持**: 完美支持自定义 TTF/OTF 字体文件。

### 5. 批量处理与预设

- **批量转换**: 支持一次性导入文件夹或多张图片进行批量处理。
- **预设管理**: 支持保存当前所有参数为预设文件 (.agp)，方便快速复用不同风格的配置。

## 版本更新 (V1.01)

- **新增**: 支持 3:2 和 2.35:1 宽高比。
- **新增**: 背景模糊亮度调节功能。
- **优化**: 水印位置升级为九宫格选择。
- **修复**: 修复了资源字体加载失败的问题。
- **还原**: 水印核心渲染逻辑回归 V1.0 稳定版，确保稳定性。

## 安装与运行

1.  确保安装 Python 3.8+。
2.  安装依赖: `pip install -r requirements.txt`
3.  运行程序: `python adaptive_glass/main.py`
