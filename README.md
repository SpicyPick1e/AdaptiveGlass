# 自适应图片比例边框处理软件 (Adaptive Glass) 

Adaptive Glass 是一款智能图片处理工具，专为摄影师和社交媒体内容创作者设计。它能自动识别图片内容，智能填充模糊背景以适应目标比例（如 Instagram 1:1, YouTube 16:9），并提供专业的水印和边框效果。

## 主要功能

### 1. 🖼️ 智能比例适配 (Smart Aspect Ratio)

- **自适应填充**: 自动计算并生成模糊背景。
- **多比例支持**:
  - **社交媒体**: 1:1 (Instagram), 4:3 (小红书), 9:16 (抖音/TikTok), 16:9 (Bilibili/YouTube).
  - **摄影/电影**: 3:2 (全画幅标准), 2.35:1 (宽银幕电影感).
- **内容缩放**: 支持 50% - 100% 主体缩放。

### 2. 💧 高级背景处理 (Advanced Background)

- **动态模糊**: 高质量高斯模糊算法。
- **风格化**: 标准模糊 / 暗色玻璃 / 亮色玻璃。
- **亮度控制**: 支持背景亮度微调 (-100 到 +100)，营造完美氛围。

### 3. ✒️ 智能水印系统 (Smart Watermark V1.02)

- **EXIF 自动读取**: 自动提取相机型号、光圈、快门、ISO、焦距。
- **九宫格定位**: 支持水印在画面 9 个方位（上/中/下 x 左/中/右）自由布局。
- **智能排版**:
  - **Append 模式**: 相机型号 + 参数 + 自定义文本。
  - **Replace 模式**: 仅显示自定义文本。
- **排版优化**: 强制底部基线对齐 (Fixed Baseline Alignment)，解决大小字号混排抖动问题。
- **自定义字体**: 支持加载本地 TTF/OTF 字体。

### 4. 🎨 精致边框与预设 (Borders & Presets)

- **边框样式**: 无边框 / 细边框 / 圆角边框，支持颜色、宽度、圆角、阴影调节。
- **预设管理**: 保存当前所有参数为 `.agp` 预设文件，一键复用风格。
- **批量处理**: 支持文件夹导入，一键批量转换海量图片。

### 5. 批量处理与预设

- **批量转换**: 支持一次性导入文件夹或多张图片进行批量处理。
- **预设管理**: 支持保存当前所有参数为预设文件 (.agp)，方便快速复用不同风格的配置。
